# ✈️ AI Travel Planner

An AI-powered travel itinerary generator. Enter any destination, select your trip duration and interests — get a personalized day-by-day plan instantly.

## Tech Stack

- Next.js 16 · TypeScript · Tailwind CSS
- Google Gemini API

## Getting Started

```bash
git clone https://github.com/kemalmustafa/travel-planner.git
cd travel-planner
npm install
```

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_key_here
```

Get your free API key at [aistudio.google.com](https://aistudio.google.com)

```bash
npm run dev
```
